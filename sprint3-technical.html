8 July 2020
HTML and CSS

In order to easily understand the use of HTML and CSS let’s compare them to a car. HTML is used in the production of the car (website), while CSS is like taking your car to a workshop to get it customised with new wheels and a paint job (or pretty much any customisation you like).

Control Flow

A script is essentially a list of instructions for a program to run – it does so from top to bottom. This is called Control Flow. Another important part of control flow is loops. Loops can be created with if and while statements, where a certain piece of code is run until a condition is met. Let's take cooking a meal as a real life example of control flow. The cookbook/recipe (script) contains the instructions and you (the program) complete those instructions in the order presented. All going well you end up with a nice meal (program)!

The DOM

The ‘Document Object Model’ is an object-oriented representation of a web page where structure, style, and content can be manipulated. This is done using a scripting language such as JavaScript.

Objects and Arrays
An object is a value stored in memory, and referenced by an identifier. It can be a variable, data structure, or method. You can access object properties in two ways:

 

‘objectName.propertyName’ (dot notation) or ‘objectName[propertyName]’ (bracket notation).

 

An array is a data type which stores a fixed number of variables which are of the same type. You can access variables within an array using bracket notation. Variables are stored in an array numerically; the first variable is denoted by 0, like so:

 

Array[0]
